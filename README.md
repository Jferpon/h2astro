## 🚀 Estructura del proyecto 

En este protecto se sigue la siguiente estructura:

```text
├── public/
├── src/
│   ├── components/
│   ├── content/
│   ├── layouts/
│   └── pages/
├── astro.config.mjs
├── README.md
├── package.json
└── tsconfig.json
```



Cualquier imagen se añade a la carpeta public o si es una portada a assets

## Comandos

Estos comandos son muy utiles de cara al proyecto 

| `npm install`             | Instala dependencias                            
| `npm run dev`             | Lanza el proyecto en el servidor localhost:4321      
| `npm run build`           | Buildea el proyecto          
| `npm run preview`         | Te permite previsualizar el buildeo antes de hacer el deploy    
| `npm run astro ...`       | Runea comandos de cliente como  `astro add`, `astro check` 
| `npm run astro -- --help` | Da ayudas con los comandos
